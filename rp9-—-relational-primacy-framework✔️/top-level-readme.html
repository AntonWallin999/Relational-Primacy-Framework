Top-level README

This repository contains the complete, final public release of
RP9 ‚Äî Relational Primacy Framework (v1.0).

RP9 is a single-axiom relational framework that specifies a closed causal origin
while allowing open systemic consequences.

No objects, forces, dimensions, measures, or empirical assumptions are postulated.

Core properties

  • Axiom count: exactly one (Axiom R0)
  • Causal status: closed at the foundation
  • Systemic consequences: open and non-restrictive
  • Derivations: none at ground level
  • Review status: methodologically locked

RP9 does not compete with existing theories or frameworks.
It specifies the condition under which any framework can function at all.

Repository structure (authoritative record)

This repository constitutes the frozen reference archive of RP9.

It contains:

  • the single axiom (LEVEL 0)
  • formal boundary and classification documents (non-ground)
  • reviewer instructions and scope lock
  • appendices (interpretative, non-causal)
  • a complete AI-based axiomatic stress test
  • license and contribution policy

Nothing outside this repository is required to review RP9 formally.
